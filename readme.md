# CareerLens

CareerLens is a full‑stack web application that helps job seekers track and manage their job applications in one place.  
Users can log applications, monitor progress through different stages (Applied, Interview, Offer, Rejected), and view insights through a visual dashboard.

This project was built as a portfolio application to demonstrate real‑world frontend and backend development skills.

---

## ✨ Features

- Secure user authentication (email & password)
- Create, update, and delete job applications
- Track application status through each hiring stage
- Dashboard with application statistics and progress insights
- Search applications by company or role
- Fully responsive UI (desktop, tablet, and mobile)
- User‑scoped data with database‑level security

---

## 🧑‍💻 Tech Stack

**Frontend**
- React
- TypeScript
- Vite
- CSS (custom design system)

**Backend**
- Supabase (Authentication & PostgreSQL database)
- Row Level Security (RLS) for data isolation
